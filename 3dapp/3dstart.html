3D on the web - The joys of three.js

Virtual reality is coming to the web in the form of Mozilla WebVR with browser support being developed for Firefox, Chrome and Edge. Today, 3D graphics in browser is made available by the THREE.js Javascript library. Both WebVR and THREE.js are based on WebGL, a browser port of the venerable OpenGL.

These libraries, vastly improve the time to code 3D applications by hiding the many details needed to render even a simple object in WebGL. No understanding of affine transforms in homogeneous coordinates necessary! No need to write code for GPU shaders!
